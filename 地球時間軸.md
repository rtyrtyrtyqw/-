 ```timeline


 + 寒武紀（Cambrian）  
 + 5.41億年前至4.85億年前
 + 標誌生物：
     寒武紀生命大爆發：在寒武紀，生命突然多樣化，出現了大量的多細胞生物，如三葉蟲、海綿動物、腕足動物、棘皮動物等。這一時期的生物擁有礦化外骨骼（如貝殼和甲殼）。

 + 奧陶紀（Ordovician）  
 + 4.85億年前至4.44億年前
 + 標誌生物：
   筆石（Graptolites）：一種浮游生物群體，是奧陶紀重要的指標化石。
   腕足動物和三葉蟲繼續繁榮，並在海洋中成為主導物種。
   出現了第一批脊椎動物，如無頜魚類，標誌著早期魚類的出現。
   海洋植物：如紅藻、綠藻等在奧陶紀繁盛。
  
   大事件：  
     ==奧陶紀志留紀大滅絕==（OrdovicianSilurian Extinction）：約4.44億年前，奧陶紀末期的滅絕事件是地球歷史上五大滅絕事件之一，估計80%以上的海洋物種消失。這與冰期和海平面下降有關。



 + 泥盆紀（Devonian）  
 + 4.19億年前至3.59億年前
 + 標誌生物：
   被稱為“魚類的時代”，泥盆紀見證了魚類的高度多樣化，包括了盔甲魚、軟骨魚類、和硬骨魚類。
   早期兩棲類開始登陸，並且是四足動物的祖先。
   陸地植物：大規模的植物演化，包括出現了有維管束的植物如蕨類植物和早期的種子植物。
  
  大事件：  
   ==泥盆紀末期大滅絕==（Late Devonian Extinction）：大約3.59億年前，這是五大滅絕事件之一，估計約75%的海洋物種消失，可能與氣候變冷和海洋缺氧有關。



 + 石炭紀（Carboniferous）  
 + 3.59億年前至2.99億年前
 + 標誌生物：
   大型森林：石炭紀是巨大蕨類植物、楔葉植物、和早期的針葉植物繁盛的時期，這些植物的遺骸形成了現代的煤層。
   昆蟲多樣化，包括巨型蜻蜓和早期的蟑螂。
   早期爬行類動物開始出現。
  




 + 二疊紀（Permian）  
 + 2.99億年前至2.52億年前
 + 標誌生物：
   合弓類動物出現，這類動物是哺乳動物的祖先。
   爬行類動物進一步繁榮，並有早期的恐龍前身出現。
   種子植物、針葉植物在陸地上繁榮。
  
   大事件：  
   ==二疊紀三疊紀大滅絕==（PermianTriassic Extinction）：2.52億年前，史上最大的大滅絕事件，超過95%的海洋物種和70%的陸地物種消失，原因可能與火山活動、氣候變遷和海洋缺氧有關。



 + 三疊紀（Triassic）  
 + 2.52億年前至2.01億年前
 + 標誌生物：
   恐龍在三疊紀中晚期開始出現，成為陸地的主導動物。
   早期哺乳類小型、夜行性動物出現。
   裸子植物，如針葉樹，在陸地上繁盛。
  
   大事件：  
   ==三疊紀侏羅紀大滅絕==（TriassicJurassic Extinction）：約2.01億年前，三疊紀末期的滅絕事件可能與火山活動和氣候變化有關，消滅了大量的爬行動物，為恐龍的崛起鋪平道路。



 + 白堊紀（Cretaceous）  
 + 1.45億年前至6600萬年前
 + 標誌生物：
   恐龍繼續在白堊紀中繁榮。
   被子植物（開花植物）開始出現並迅速擴散。
   早期哺乳動物：一些小型的有袋類和胎盤哺乳動物出現。
  
   大事件：  
   ==白堊紀第三紀大滅絕==（CretaceousPaleogene Extinction）：6600萬年前，小行星撞擊地球和火山活動導致恐龍滅絕，75%的物種消失，標誌著哺乳動物崛起。
```
