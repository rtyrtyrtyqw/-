```timeline
+ 冥古宙（Hadean Eon）
+ 約45億年前 – 40億年前
+ 大氣特徵：冥古宙時期的大氣以火山活動釋放的氣體為主，這些氣體包括氫氣、氦氣、甲烷（CH₄）、氨（NH₃）、水蒸氣（H₂O）和二氧化碳（CO₂）。大氣幾乎不含氧氣（O₂）。
  標記生物：這一時期尚無已知的生物，地球處於極端高溫且劇烈的火山活動中，生命可能尚未出現或僅有早期的有機化學反應。
+ 太古宙（Archean Eon)
+ 約40億年前 – 25億年前
+ 大氣特徵：大氣仍然缺乏自由氧氣，主要由二氧化碳、甲烷、氨和其他氣體組成。甲烷和二氧化碳可能導致了當時的溫室效應，保持了地球的溫暖。
	標記生物：主要是原核生物，尤其是古菌和厭氧細菌。這些生物能夠在無氧環境下生存並進行厭氧代謝。這一時期的標記生物是化能自營細菌，它們能夠利用化學能合成有機物。
  疊層石：由藍綠菌（也稱藍藻）構成的疊層石是這一時期的重要化石證據。藍綠菌進行光合作用，這為大氣中氧氣的逐漸積累奠定了基礎。


+ 元古宙（Proterozoic Eon）
+ 約25億年前 – 5.4億年前
+ 大氣特徵：元古宙時期見證了大氧化事件（Great Oxygenation Event, 約24億年前），大氣中的氧氣含量開始顯著上升，主要來自藍綠菌光合作用的產物。==大氣中氧氣的累積導致了氧化鐵的形成，表現在地質上為帶狀鐵層==     （Banded Iron Formations, BIFs）。
  標記生物：
  藍綠菌繼續繁衍，它們產生的氧氣改變了大氣和海洋的化學性質。
  真核生物出現：隨著氧氣濃度的增加，真核生物（具有細胞核的生物）在元古宙晚期出現。這些生物能夠利用有氧呼吸提供能量。
  多細胞生物：元古宙末期，海洋中出現了早期的多細胞生物，這包括一些簡單的軟體動物和藻類。

+ 古生代（Paleozoic Era）
+ 約5.4億年前 – 2.5億年前
+ 大氣特徵：大氣中的氧氣含量逐漸增高，尤其是在寒武紀大爆發（Cambrian Explosion）之後，氧氣濃度進一步上升。這一時期的二氧化碳濃度相對較高，但隨著植物的出現而逐漸下降。
  標記生物：
  寒武紀大爆發標誌著大量多細胞生命形式的出現，從三葉蟲到腕足動物，生命的多樣性大大增加。
  魚類、無脊椎動物以及後來的兩棲動物開始適應新的富氧環境。

+ 泥盆紀（Devonian）植物：
+ 4.19億年前至3.59億年前
+ 陸地植物（如蕨類植物）開始廣泛繁殖，這些植物通過光合作用吸收二氧化碳並釋放氧氣，進一步改變了大氣成分，促進了氧氣含量的增加。

+ 中生代（Mesozoic Era）
+ 約2.5億年前 – 6600萬年前
+ 大氣特徵：這一時期大氣中的氧氣含量基本穩定，但二氧化碳濃度相對較高，導致全球溫度較高。這一時期見證了恐龍的繁盛和大型爬行動物的出現。
標記生物：
爬行動物和恐龍：中生代尤其是侏羅紀（Jurassic）和白堊紀（Cretaceous）標誌著恐龍的盛世，它們適應了溫暖、濕潤的氣候。
被子植物：在白堊紀晚期，被子植物（開花植物）開始出現並繁榮，這進一步改變了大氣中的氧氣和二氧化碳比例，並為昆蟲和其他動物提供了新的食物來源。

+ 新生代（Cenozoic Era）
+ ,約6600萬年前至今
+ 大氣特徵：氧氣含量接近現代水平，二氧化碳濃度逐漸下降，特別是在第四紀的冰河期，二氧化碳含量顯著降低。這一時期的氣候波動也影響了物種的演化。
標記生物：
哺乳動物：新生代早期，隨著恐龍的滅絕，哺乳動物迅速繁盛，特別是靈長類動物，它們適應了富氧的環境。
被子植物和草原生態系統：草原和其他植被覆蓋廣泛，進一步改變了大氣中的氣體組成，為草食性動物和肉食性動物提供了食物鏈的基礎。
```
